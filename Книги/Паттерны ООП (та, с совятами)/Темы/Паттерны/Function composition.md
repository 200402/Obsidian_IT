Композиция функций (суперпози́ция фу́нкций) — это применение одной функции как параметр другой.

---
| До                     | после:                                    |
| ---------------------- | ----------------------------------------- |
| const x1 = 1;          | const result = add(3, add(2, add(1, 1))); |
| const x2 = add(x1, 1); |                                           |
| const x3 = add(x1, 1); |                                           |
| const x4 = add(x1, 1); |                                           |
| const result = x4;     |                                           |
